View your app in AI Studio: https://ai.studio/apps/e68eb219-f401-48f6-89a4-9a14301fa8b9

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
